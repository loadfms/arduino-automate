# Automate

Simple project to turn on and off any thing connected to circuits relays.

## Components
- Arduino Uno
- Ethernet card ENC28J60
- Relays
- Protoboard

## Library
Included in project.

## How it works
Arduino execute a request to api and it responds if that relay should be open or not.
